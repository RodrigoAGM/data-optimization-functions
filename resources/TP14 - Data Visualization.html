La visualización de datos es la disciplina de tratar de comprender los datos colocándolos en un contexto visual para que los patrones, las tendencias y las correlaciones no puedan detectarse.

Python ofrece múltiples bibliotecas gráficas excelentes que vienen con muchas características diferentes. No importa si desea crear Python interactivo, en vivo o altamente personalizado, tiene una excelente biblioteca para usted.

Para obtener una breve descripción general aquí hay algunas bibliotecas de visualizacion populares:

  • Matplotlib: nivel bajo, proporciona mucha libertad.
  • Visualización de pandas: interfaz fácil de usar, construida en Matplotlib
  • Seaborn: interfaz de alto nivel, excelentes estilos por defecto
  • ggplot: basado en ggplot2 de R, utiliza gramática de gráficos
  • Plotly: puede crear parcelas interactivas

Aquí, aprenderemos cómo crear diagramas básicos utilizando Matplotlib, visualización de Pandas y Seaborn, así como a usar algunas características específicas de cada biblioteca.

Matplotlib¶

Matplotlib es la biblioteca de visualizacionde python más popular. Es una biblioteca de bajo nivel libre. Matplotlib is specifically good for creating basic graphs like line charts, bar charts, histograms and many more. It can be imported by typing:

In [15]:
import matplotlib.pyplot as plt

Scatter Plot¶

Para crear un diagrama de dispersión en Matplotlib podemos usar el método de dispersión. También crearemos una figura y un eje utilizando plt.subplots para que podamos asignar a nuestra parcela un título y etiquetas.

In [17]:
# create a figure and axis
fig, ax = plt.subplots()

# scatter the sepal_length against the sepal_width
ax.scatter(iris['sepallength'], iris['sepalwidth'])
# set a title and labels
ax.set_title('Iris Dataset')
ax.set_xlabel('sepallength')
ax.set_ylabel('sepalwidth')
Out[17]:
Text(0, 0.5, 'sepalwidth')

Podemos dar el gráfico coloreando cada punto de datos por su clase. Esto se puede hacer creando un diccionario que le permita familiarizarse entre sí.

In [18]:
#create color dictionary
colors = {'Iris-setosa':'r', 'Iris-versicolor':'g', 'Iris-virginica':'b'}

fig, ax = plt.subplots()

for i in range(len(iris['sepallength'])):

Gráfico de líneas¶

En Matplotlib podemos crear un gráfico de líneas llamando al método de trazado. También podemos trazar varias columnas en un gráfico, haciendo un bucle a través de las columnas que queremos trazar cada columna en el mismo eje.

In [19]:
# get columns to plot
columns = iris.columns.drop(['class'])
# create x data
x_data = range(0, iris.shape[0])
# create figure and axis
fig, ax = plt.subplots()
# plot each column
for column in columns:
    ax.plot(x_data, iris[column])
# set title and legend
ax.set_title('Iris Dataset')
ax.legend()

Gráfico de barras¶

Se puede crear un gráfico de barras utilizando el método de barra. El gráfico de barras no calcula automáticamente la frecuencia de una categoría de cosas. El gráfico de barras es útil para muchas categorías diferentes (menos de 30) porque puede ser bastante complicado.

In [21]:
# create a figure and axis 
fig, ax = plt.subplots() 
# count the occurrence of each class 
data = wine_reviews['points'].value_counts() 
# get x and y data 
points = data.index 
frequency = data.values 
# create bar chart 
ax.bar(points, frequency) 
# set title and labels 
ax.set_title('Wine Review Scores') 
ax.set_xlabel('Points') 
ax.set_ylabel('Frequency')

Visualización en pandas¶

Pandas es una biblioteca de código abierto de alto rendimiento y fácil de usar que proporciona estructuras de datos, como marcos de datos y herramientas de análisis de datos como las herramientas de visualización que usaremos en este artículo.

La visualización de pandas hace que sea realmente fácil crear un marco de datos y una serie de pandas. También tiene un nivel de API más alto que Matplotlib y, por lo tanto, necesitamos menos código para los mismos resultados.

Parcela de dispersión¶

Para crear un diagrama de dispersión en Pandas podemos llamar a .plot.scatter () y pasarle dos argumentos, el nombre de la columna x y el nombre de la columna y. Opcionalmente también podemos pasarle un título.

In [23]:
iris.plot.scatter(x='sepallength', y='sepalwidth', title='Iris Dataset')
Out[23]:
<matplotlib.axes._subplots.AxesSubplot at 0x7fdc81516cf8>

Como se puede ver en la imagen, está configurando automáticamente las etiquetas x e y a los nombres de columna.

Gráfico de líneas¶

Para crear un gráfico de líneas en Pandas podemos llamar .plot.line (). Mientras que en este caso necesitamos recorrer cada columna que queremos trazar, en la que no necesitamos especificar una (s) columna (s) específica (s).

Seaborn¶

Seaborn es una biblioteca de visualización de datos de Python basada en Matplotlib. Proporciona una interfaz de alto nivel para crear gráficos atractivos.

Seaborn tiene mucho que ofrecer. Puede crear gráficos en una línea que le llevaría a múltiples líneas de líneas en Matplotlib. Sus diseños estándar son asombrosos y también tiene una interfaz agradable para trabajar con marcos de datos pandas.

Se puede importar escribiendo:

In [30]:
import seaborn as sns

Mapa de calor¶

Un mapa de calor es una representación gráfica de datos donde los valores individuales contenidos en una matriz se representan como colores. Los mapas de calor son perfectos para explorar la correlación de características en un conjunto de datos.

Para obtener la correlación de las características dentro de un conjunto de datos, podemos llamar .corr (), que es un método de marco de datos de Pandas. Esto nos dará la matriz de correlación.

Ahora podemos usar Matplotlib o Seaborn para crear el mapa de calor.

Matplotlib:

In [39]:
import numpy as np
# get correlation matrix
corr = iris.corr()
fig, ax = plt.subplots()
# create heatmap
im = ax.imshow(corr.values)

# set labels
ax.set_xticks(np.arange(len(corr.columns)))
ax.set_yticks(np.arange(len(corr.columns)))
ax.set_xticklabels(corr.columns)
ax.set_yticklabels(corr.columns)

# Rotate the tick labels and set their alignment.
plt.setp(ax.get_xticklabels(), rotation=45, ha="right", rotation_mode="anchor")
Out[39]:
[None, None, None, None, None, None, None, None]

ax.set_yticklabels(corr.columns)

# Rotate the tick labels and set their alignment.

        text = ax.text(j, i, np.around(corr.iloc[i, j], decimals=2),
                       ha="center", va="center", color="black")

Facetado¶

Facetar es el acto de dividir las variables de datos en múltiples subparcelas y combinarlas en una sola figura.

Facetar es realmente útil si desea explorar rápidamente su conjunto de datos.

Para usar un tipo de facetado en Seaborn podemos usar FacetGrid. En primer lugar, debemos definir el FacetGrid y pasarle nuestros datos, así como una fila o columna, que se utilizará para dividir los datos. Luego debemos llamar a la función de mapa en nuestro objeto FacetGrid y definir el tipo de gráfico que queremos usar, así como la columna que queremos graficar.

In [43]:
g = sns.FacetGrid(iris, col='class')
g = g.map(sns.kdeplot, 'sepallength')
