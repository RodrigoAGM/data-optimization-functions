La visualizacin de datos es la disciplina de tratar de comprender los datos colocndolos en un contexto visual para que los patrones, las tendencias y las correlaciones no puedan detectarse.

Python ofrece mltiples bibliotecas grficas excelentes que vienen con muchas caractersticas diferentes. No importa si desea crear Python interactivo, en vivo o altamente personalizado, tiene una excelente biblioteca para usted.

Para obtener una breve descripcin general aqu hay algunas bibliotecas de visualizacion populares:

  • Matplotlib: nivel bajo, proporciona mucha libertad.
  • Visualizacin de pandas: interfaz fcil de usar, construida en Matplotlib
  • Seaborn: interfaz de alto nivel, excelentes estilos por defecto
  • ggplot: basado en ggplot2 de R, utiliza gramtica de grficos
  • Plotly: puede crear parcelas interactivas

Aqu, aprenderemos cmo crear diagramas bsicos utilizando Matplotlib, visualizacin de Pandas y Seaborn, as como a usar algunas caractersticas especficas de cada biblioteca.

Mapa de calor¶

Un mapa de calor es una representacin grfica de datos donde los valores individuales contenidos en una matriz se representan como colores. Los mapas de calor son perfectos para explorar la correlacin de caractersticas en un conjunto de datos.

Para obtener la correlacin de las caractersticas dentro de un conjunto de datos, podemos llamar .corr (), que es un mtodo de marco de datos de Pandas. Esto nos dar la matriz de correlacin.

Ahora podemos usar Matplotlib o Seaborn para crear el mapa de calor.

Matplotlib:

In [39]:

# get correlation matrix
corr = iris.corr()
fig, ax = plt.subplots()
# create heatmap
im = ax.imshow(corr.values)

# set labels
ax.set_xticks(np.arange(len(corr.columns)))
ax.set_yticks(np.arange(len(corr.columns)))
ax.set_xticklabels(corr.columns)
ax.set_yticklabels(corr.columns)

# Rotate the tick labels and set their alignment.
plt.setp(ax.get_xticklabels(), rotation=45, ha="right", rotation_mode="anchor")

Facetado¶

Facetar es el acto de dividir las variables de datos en mltiples subparcelas y combinarlas en una sola figura.

Facetar es realmente til si desea explorar rpidamente su conjunto de datos.

Para usar un tipo de facetado en Seaborn podemos usar FacetGrid. En primer lugar, debemos definir el FacetGrid y pasarle nuestros datos, as como una fila o columna, que se utilizar para dividir los datos. Luego debemos llamar a la funcin de mapa en nuestro objeto FacetGrid y definir el tipo de grfico que queremos usar, as como la columna que queremos graficar.
