Cuantificación¶

Cuantificacin es una tcnica de compresin con prdida que consiste en agrupar todo un rango de valores en uno solo. Si cuantificamos el color de una imagen, reducimos el nmero de colores necesarios para representarla y el tamao del fichero de la misma disminuye. Esto es importante, por ejemplo, para representar una imagen en dispositivos que slo dan soporte a un nmero limitado de colores.

La funcin pairwise_distances_argmin https://scikit-learn.org/stable/modules/generated/sklearn.metrics.pairwise_distances_argmin.html#sklearn.metrics.pairwise_distances_argmin calcula los ndices de los puntos ms cercanos.

Función que recrea una imagen de las etiquetas del libro de códigos.¶

In [11]:
def recreate_image(codebook, labels, w, h):
    d = codebook.shape[1]
    image = np.zeros((w, h, d))
    label_idx = 0
    for i in range(w):
        for j in range(h):
            image[i][j] = codebook[labels[label_idx]]
            label_idx += 1
    return image
